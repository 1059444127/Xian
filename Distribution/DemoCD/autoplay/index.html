Welcome to the ClearCanvas Workstation demonstration CD-ROM. This CD contains the installation program for the beta version 0.9 of Workstation, as well as a host of resources. Here, you can learn how to build the software from scratch using a tool like Visual C# Express, how the software works and how the pieces fit together, and even how to write plugins and contribute to this Open Source project.

For more information, please contact us.

--= Shortcuts =--

Install CC Workstation Beta This launches a self-contained installation program that will install the Workstation beta software onto your computer. This software is for evaluation purposes only and should not be used in a clinical setting. This version of the software requires Windows 2000 or later with Microsoft .NET 2.0 in order to run.

Sample Images This is a collection of sample DICOM images that you may use with CC Workstation to test out the currently available features. To make use of these sample images, you need to run this self-extracting archive and store the images to a folder on your local harddisk. Then you may either use the "My Computer" Study Explorer widget to load the images for viewing, or use the Rebuild Utility to put references to the images into a SQL Server database. If the images are referenced in the database, you may them load them from the "My Local DataStore" widget in the future.

Install SQL Server 2005 Express Edition CC Workstation is currently able to load images for viewing in two different ways. First, you can browse your local filesystem using a file browser and pick the images you want to load. This is done through the "My Computer" widget. In addition, however, you may opt to use SQL Server to persist references to the images stored on disk. This means you won't have to use a browser every time you want to take a look at some images. Image loading performance is also a bit better with a database. This second way of loading images is done through the "My Local DataStore" widget. You must install SQL Server 2005 Express Edition manually and select the SQL Server Authenticatoin type. You must then run a set of SQL scripts that will create the database schema for you.

SDK Documentation This is comprised of two documents: a developer's guide, which you can view if you have a PDF reader, and the Framework Reference.

Source Code The source tree is archived into a ZIP file. To start working with the source, simply extract the files into their own folder on your local harddisk drive. To build the source code, you will need a C# compiler like Visual Studio 2005, or Visual C# Express Edition, which you may find on this CD-ROM.

Download Visual C# Express Edition What a boon from Microsoft! It's now easier than ever to create your own software or learn to write software programs when Microsoft provides this edition of their C# compiler and development environment for free.

Visual Studio Templates You can install and use these templates to get a head start in writing plugins for ClearCanvas Workstation. We hope you will enjoy this convenience and have fun with it!
